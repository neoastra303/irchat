If you want to enable PyPI publishing:
  1. Create/obtain PyPI API token from https://pypi.org/account/
  2. Add secret: echo "YOUR_TOKEN" | gh secret set PYPI_API_TOKEN
  3. Uncomment publish-pypi.yml workflow

If you want to enable Codecov:
  1. Sign up at https://codecov.io
  2. Get repo token
  3. Add secret: echo "YOUR_TOKEN" | gh secret set CODECOV_TOKEN

If you want GitHub Container Registry (GHCR) publishing:
  1. Create GitHub PAT with read/write packages scope
  2. Add secret: echo "YOUR_TOKEN" | gh secret set GHCR_PAT
  3. Uncomment GHCR publish step in docker-build.yml
